HTML

The html tag is used to tell the computer that the following document is going to be written in HTML. Its syntax usually at the very beginning and end of the and is usually represented by a <html> tag. This element is used in all html documents and is always at the beginning and end of all the other code,but it never appears on your web page when you use it.

An example

<html>
*insert random code
</html>


Headers

The header tag is used to tell the computer to not include the following information as actual rendering but instead tells the computer use the data included inside to describe text that will appear on the website. Its syntax is <head>, and it appears usually right after the first html tag and after all the data decribing the document's style. It is also used in all html docs with a style description not included in the paragraph or heading tags.


An example

<html>
<head>
*Random Stuff
</head>
...


Titles

Headings

Headings are the tags for the large bold text that I use to start out my paragraphs with a title. It creates the title of "headings" for this paragraph. Its syntax is <h1>, but can be any number from 1-6

Paragraphs

Paragraphs are the backbone of creating a website. Most of the text you write will be in this format. Paragraphs' syntaxes are represented by the <p> tag. They are basically the small text that I am using to write all this text with.


An example

Breaks

Breaks are the same as a 'return/enter' when you are typing. It just skips your text to the next line. Their syntax is <br>. I use breaks to separate my paragraphs to organise them so they aren't combined in one giant paragraph. There are mainly used for aesthetics as well, and I use this almost 40 times in just this reference sheet.


An example

text
<br>
text

Comments

Comments are used to write random stuff in your code for reference without them actually appearing in your actual site. They are represented by the <!-- --> tag. You insert whatever you comment in between the dual <--'s . Comments are useful for people while they are making their website to remind themselves to do stuff later.


An example


text
<!--fsdffds-->.
Note that the comment text should not actually appear


Links

Links are a very important part of a web page, as you can link them to another web site by a single click. Links follow the <a href="...">...> structure. The first ... is the actual url of the link and the second... is the text it appears as. I use this kind of code to link my ref sheet to my main page.


An example as well as appearance


<a href="www.google.com">Google Website>

Then you put the stuff you want to style in between the beginning and end of the tags. This would be useful for giving a website multiple different sections with different styles.
